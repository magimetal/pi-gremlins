# Notice

`pi-gremlins` is forked from and inspired by [`nicobailon/pi-subagents`](https://github.com/nicobailon/pi-subagents).

This repository adapts that idea into gremlin-themed branding and local standalone package/repo structure for GitHub-based Pi installation.

Core concept retained:

- delegate work to isolated Pi subprocesses
- support single, parallel, and chain execution flows
- use Pi agent definitions from standard agent directories

Branding, packaging, documentation, and standalone development setup in this repository are maintained separately here.
